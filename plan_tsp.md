# Plan: Hybrid TSP Solver (Genetic Algorithm + Simulated Annealing)

## 1. Project Goal
Develop a Python program to solve the Traveling Salesman Problem (TSP) using a Hybrid Algorithm. The system combines a Genetic Algorithm (GA) for global search with Simulated Annealing (SA) for local optimization of children.

## 2. Core Components

### A. Data Structures
*   **`City` Class**: Represents a location with $(x, y)$ coordinates. Includes distance calculation methods.
*   **`Route`**: A list of `City` objects representing a path.
*   **`Population`**: A collection of random routes.

### B. Genetic Algorithm (The Framework)
1.  **Selection (Tournament)**:
    *   Randomly select a subset of the population (e.g., size 5).
    *   The fittest individual (shortest distance) in the subset becomes a parent.
2.  **Crossover (Ordered Crossover - OX1)**:
    *   Essential for TSP to avoid visiting cities twice.
    *   Select a sub-sequence from Parent 1.
    *   Fill remaining slots with cities from Parent 2 in the order they appear, skipping those already in the sub-sequence.
3.  **Mutation (Swap)**:
    *   With low probability, swap two cities in the route.
4.  **Elitism**:
    *   The single best route from the previous generation is passed directly to the new generation without changes.

### C. Simulated Annealing (The Improver)
*   **Role**: Applied to *children* generated by the GA to refine them before they enter the new population.
*   **Process**:
    *   Start with the child route.
    *   make small changes (swap neighbors).
    *   Accept improvements always; accept worse solutions with probability $P = \exp(-\Delta E / T)$.
    *   Cool down rapidly (since this runs many times).

## 3. Execution Flow (Pseudocode)

```python
Initialize Population
Best_Route = None

For Generation in 1 to MAX_GENERATIONS:
    # 1. Elitism
    New_Population.add(Best_Route_From_Previous_Gen)

    # 2. Breeding Loop
    While New_Population is not full:
        Parent1 = Tournament_Selection(Population)
        Parent2 = Tournament_Selection(Population)

        Child = Crossover(Parent1, Parent2)

        if random < Mutation_Rate:
            Mutate(Child)

        # 3. Hybrid Step: Local Optimization
        Child = Simulated_Annealing(Child, Temp=100, Cooling=0.9)

        New_Population.add(Child)

    Population = New_Population
```
